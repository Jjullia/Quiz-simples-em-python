# ❓ Quiz Simples em Python

## 📌 Sobre o Projeto

Este projeto consiste em um quiz simples desenvolvido em Python com o objetivo de praticar lógica de programação, estruturas condicionais e interação com o usuário no terminal.

A aplicação apresenta perguntas e respostas, valida as escolhas do usuário e exibe a pontuação final ao término do quiz.

---

# 🎯 Objetivos

* Praticar lógica de programação;
* Trabalhar com condicionais e estruturas de repetição;
* Desenvolver interação com o usuário via terminal;
* Criar um sistema simples de perguntas e respostas em Python.

---

# 🛠️ Tecnologias Utilizadas

* Python
* VS Code / Jupyter Notebook

---

# ⚙️ Funcionalidades

O projeto permite:

* Exibir perguntas no terminal;
* Receber respostas do usuário;
* Validar respostas corretas e incorretas;
* Calcular pontuação final;
* Simular um jogo de perguntas e respostas simples.

# 🚀 Como Executar

1. Clone este repositório:

```bash

```

2. Acesse a pasta do projeto:

```bash
cd Quiz-simples-em-python
```

3. Execute o arquivo Python:

```bash
python quiz.py
```
